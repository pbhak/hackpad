# hackpad
A custom-built macropad made for Hackpad, a Hack Club YSWS

## Ideas of stuff to map to
- IDE tools
  * Terminal, Start/Stop, Restart, etc
- Common shortcuts
  * Copy, Paste, Cut
- Applications
  * Open Terminal, Chrome window, etc
- Layer switching
- Encoder
  * Scroll, zoom, alt-tab (my main keyboard has vol already)
